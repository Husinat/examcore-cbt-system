# ExamCore CBT System

**ExamCore** is a modern Computer-Based Testing (CBT) platform designed to provide students with a realistic exam practice environment.  
It simulates real exam conditions, allowing users to practice, track performance, and improve scores efficiently.

## Features

- **Realistic Exam Interface** – Navigate through questions like a real CBT terminal, with time tracking.  
- **Timed & Secure Testing** – Simulated exam duration with secure data handling.  
- **Analytics & Insights** – Instant performance reports and progress tracking.  
- **Responsive Design** – Works seamlessly on desktop and mobile devices.  
- **User Authentication** – Secure login and signup powered by Firebase.  


## How It Works

1. **Sign Up / Login** – Create an account from the landing page to start practicing.  
2. **Choose Exam** – Select your subject combination and available mock exams.  
3. **Take Test** – Complete the timed mock exam and receive instant results.  
4. **Review Performance** – Check detailed analytics and track your progress over time.


## Tech Stack

- **Frontend:** HTML, CSS, TailwindCSS, JavaScript
- **Backend / Database:** Firebase Authentication & Firestore  
- **Tools:** GitHub for version control and project management
